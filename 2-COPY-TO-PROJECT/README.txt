STEP 2 — COPY THESE FILES TO YOUR PROJECT
==========================================
The files you need depend on which tool you use.

ALL TOOLS — copy these 2 files to your project root:
  CONTEXT.md       <- your living project brain
  PROMPT-LOG.md    <- your self-improving prompt log

THEN, depending on your tool:

V0.APP:
  Copy AI_RULES.md to your project root.
  Push all 3 files to your GitHub repo.
  V0 reads them from the connected GitHub repo by filename.

DYAD.SH:
  Copy AI_RULES.md to your project root folder.
  (Overwrites Dyad's default AI_RULES.md -- click YES)
  Dyad reads AI_RULES.md automatically every chat.

WINDSURF (Windows/Mac/Linux desktop IDE):
  Copy .windsurfrules to your project root folder.
  (This is the native Windsurf rules filename)
  Cascade reads .windsurfrules automatically every session.
  Also copy AI_RULES.md for reference / backup.

See START-HERE.html for full setup steps for each tool.
