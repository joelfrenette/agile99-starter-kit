PV (Planned)
--
% work planned for today
Planned Value: % of work that SHOULD be done by now based on the schedule. PV = (Planned tasks by today / Total tasks) × 100
EV (Earned)
--
% work actually complete
Earned Value: % of work ACTUALLY completed. EV = (Completed tasks / Total tasks) × 100
SV (Variance)
--
EV minus PV
Schedule Variance: + = ahead, 0 = on track, - = behind. SV = EV - PV
SPI (Index)
--
efficiency vs plan
Schedule Performance Index. SPI ≥ 1.0 = on/ahead of schedule. SPI = EV / PV
Status
ON TRACK
SPI ≥ 0.9 = ON TRACK (green). SPI < 0.9 = BEHIND (red). SPI > 1.1 = AHEAD (blue).
% Complete
--
timeline elapsed
% Complete = (Days elapsed / Total days) × 100. This is time, not work.
ETC (Remaining)
--
weeks of work left
Estimate to Complete. ETC = Remaining work / SPI. Adjusts for your current pace.
